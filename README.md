# 世界史 一問一答

スマホ向けの世界史一問一答学習アプリ。**4方向スワイプ**で高速に周回しながら、**習熟度の伸びを可視化**して「繰り返すほど定着する」を実感できる。ログイン不要・オフライン対応・進捗は端末内に保存。

> **ステータス:** 🚧 設計・計画フェーズ完了 / 実装はこれから
> 設計と実装計画は [`docs/`](docs/) にまとまっています。

---

## 特長

- **4方向スワイプ回答**（スマホ最適・片手操作）
  - → 覚えた / ← できなかった / ↑ あやふや / ↓ ドラッグで解答を"絞り"表示（バカラの絞りのイメージ）
- **4段階の習熟度**（未回答 / できなかった / あやふや / マスター）。「覚えた」を連続2回でマスター
- **習熟マップ**：全27章を習熟度カラーで俯瞰、100%達成で 🏆 トロフィー
- **章ごとの積層棒グラフ**：繰り返すほど緑（マスター）が積み上がる伸びを可視化
- **ストリーク**（連続学習日数）でモチベ維持
- **着せ替え**：ライト/ダークテーマ＋アクセントカラー（後追い）
- **PWA**：ホーム画面に追加、電波なしでも学習可能
- **進捗のバックアップ**：JSON エクスポート/インポート

---

## 画面構成（3タブ）

| タブ | 内容 |
|---|---|
| ホーム | 習熟マップ（27章のヒートマップ）・全体習熟度・あやふや復習 |
| 成績 | 全体習熟度・ストリーク・苦手な章 |
| 設定 | テーマ/アクセント・出題順・セッション問題数・データ管理 |

このほか、章詳細（伸びグラフ）／出題（フラッシュカード）／リザルトの没入画面がある。

## 技術構成

- **Vite + React + TypeScript**（静的SPA）
- ルーティング: **React Router（HashRouter）** … GitHub Pages でのリロード404を回避
- 状態/保存: React Context + **localStorage**（サーバー・ログインなし）
- グラフ: 依存を増やさず **inline SVG** で自作
- テスト: **Vitest + Testing Library**（ドメインロジックはTDD）
- PWA: **vite-plugin-pwa**

## 開発（実装開始後）

```bash
npm install
npm run dev        # 開発サーバ（/world-history-quiz/ パスで開く）
npm test           # テスト
npm run questions  # 問題CSV→JSON変換（data/questions.csv を配置後）
npm run build      # 本番ビルド
```

### 問題データ

出典は Google スプレッドシート（`No / チャプター / 質問 / 解答` の4列・全27章・約847問）。
`ファイル > ダウンロード > CSV` で書き出して `data/questions.csv` に配置し、`npm run questions` で `src/data/questions.json` を生成する。

---

## GitHub Pages 公開（予定）

1. `npm run build` で `dist/` を生成
2. `dist/` を Pages に配信（`npx gh-pages -d dist` または GitHub Actions）
3. `vite.config.ts` の `base` はリポジトリ名（`/world-history-quiz/`）と一致させる
4. Settings > Pages で公開ブランチ/ディレクトリを指定

---

## データの扱い

学習の進捗はすべて**ブラウザ内（localStorage）に保存**され、サーバーには送信されません。
機種変更やブラウザのデータ削除で消えるため、設定の**エクスポート**で定期的にバックアップできます。
